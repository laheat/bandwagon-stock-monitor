## 一、为什么"搬瓦工库存"成了圈内常驻话题

如果你最近在搜"搬瓦工库存"，大概率不是闲着没事搜着玩。要么是看中了某款限量版套餐，跑去官网发现按钮灰着，写着一句冷冰冰的 "out of stock"；要么是听朋友说"那个 49.99 美元一年的 CN2 GIA-E 限量版又补货了"，等你打开浏览器已经售罄；要么就是被各种搬瓦工中文网、博客里的"补货通知"刷屏，想知道这些通知到底靠不靠谱、自己怎么第一时间拿到。

说白了，搬瓦工（BandwagonHost）的库存问题之所以值得专门聊，是因为它身上有个很矛盾的现象：这家公司服务稳定、口碑在线、CN2 GIA 线路在同价位里几乎没有对手，但偏偏最香的几款限量版套餐长期处于"刚补完货就秒光"的状态。常规套餐官网首页挂着随便买，限量版却要靠盯盘、靠手速、靠信息差。

这篇就把搬瓦工库存这件事从里到外捋一遍：库存怎么查、补货什么时候来、通知从哪接收、限量版都有些什么货、所有在售套餐的价格配置长什么样、怎么挑、怎么省钱、买完怎么换机房。能一次看懂的，就不必再去翻七八个站点拼凑信息了。

## 二、搬瓦工库存为什么会缺：限量版与常规版的区别

先把概念分清楚，不然后面聊补货会越聊越乱。

**常规套餐**指的是官网 VPS Hosting、CN2 GIA、香港、日本、新加坡、迪拜这些栏目下长期挂着、随时可以下单的标准方案。这些方案理论上长期有货，下单后基本是即时开通。比如 KVM 20G 年付 49.99 美元那款入门款，官网上常年挂着，闭眼买就行。

**限量版套餐**则是搬瓦工不定期推出的"特价窗口"，命名往往很放飞自我：THE PLAN、THE CHICKEN、THE PLAN v2、MINICHICKEN、The Tokyo Plan、BiggerBox、PowerBox、SAKURABOX、OpenClaw VPS……每一款都有特定的机房、配置和价格，共同特征是**便宜得离谱**，比如年付 49.99 美元的 CN2 GIA-E 10G 限量版、年付 19 美元的 MINICHICKEN、季付 49.99 美元的 DC6 CN2 GIA-E 限量版。

为什么限量？这事儿搬瓦工自己在 CN2 GIA 介绍页里说得很明白：CN2 GIA（AS4809）这种级别的 IP 中转，单 Mbps 成本可以高到 120 美元，1Gbps 满载跑一个月在某些市场要烧到约 10 万美元。**带宽是真贵，容量是真有限**，所以限量版本质上是搬瓦工在限定库存的优质线路上，划出一小批促销名额放出来。卖完了就是卖完了，得等下一批。

所以"搬瓦工库存"这件事，主要是盯限量版。常规版基本不缺货，缺了你也能切到同价位其他机房凑合用，限量版才是真要抢的东西。

## 三、查搬瓦工库存的三种姿势

### 1. 官网方案列表页直接看

最直接的办法就是去搬瓦工官网的方案页面，挨个套餐看 "Order Now" 按钮是否点亮。有货是亮色按钮，缺货会标 "out of stock"。这个办法胜在权威，但劣势也很明显：你不可能 24 小时盯屏幕，限量版补货窗口经常就一两天，纯靠人眼盯根本抢不过脚本。

### 2. 第三方实时库存监控页

圈内早就有人做了自动化监控页，把搬瓦工所有套餐的库存状态汇总到一个页面，每隔几分钟刷新一次。比较稳的有这么几个：

- **stock.bwg.net**：搬瓦工中文网维护的实时库存监控页，每 5 分钟刷新一次，所有在售套餐的库存状态一目了然，有货显示绿色，缺货标红，是当前公认最全最及时的库存看板。
- **status.bwgyhw.cn**：另一个长期维护的搬瓦工方案套餐价格配置汇总与库存监控站，可以一并参考。
- **kucun.banwagong.net**：搬瓦工补货归档类站点的实时库存版本，新版同样每 5 分钟刷新。

把这几个监控页丢到浏览器收藏栏，没事刷一刷，比每天登录官网逐个套餐点过去要省事得多。要更省事，下一节的"补货通知"才是真正适合躺平的玩法。

### 3. KiwiVM 后台查"迁移可选机房"

这是个比较少人提的小技巧。如果你已经买过搬瓦工的某款套餐，登录 KiwiVM 控制面板，进入 "Migrate to another DC"（迁移到其它数据中心）功能，会列出当前这款套餐**支持迁移的所有机房**。这个列表本身不算严格的库存查询，但能告诉你"哪些机房还有空位可迁"，反向推算某些热门机房的紧张程度。比如 CN2 GIA-E 套餐能迁的机房列表里包含 DC6 CN2 GIA-E、DC9 CN2 GIA、JPOS\_1（日本软银）、EUNL\_9（荷兰联通 9929）等，一旦某个机房突然从可迁列表里消失，基本就是满了。

## 四、搬瓦工补货规律与抢购时间窗口

补货没有公开的固定时间表，这是搬瓦工官方一直以来的态度。但圈内整理了多年的补货记录，规律还是比较明显的。

根据多位博主整理的搬瓦工补货归档，**一般节奏是一周到半个月补货一次，每次补货窗口持续 1 天到 2 天**。具体补哪款、补多少，看搬瓦工当时的带宽库存和运营节奏。最近几个月比较有代表性的补货记录包括：DC6 CN2 GIA-E 限量版 49.99 美元/年、CN2 GIA-E 10G 限量版 49.99 美元/年、MINICHICKEN 19 美元/年、The Tokyo Plan VPS、CN2 GIA VPS SG\_8（2026 年 5 月）、OpenClaw VPS（2026 年 3 月）等。

抢购的实操要点其实就三条：

1. **别犹豫**：限量版补货后通常几个小时内就会被搬空，热门款比如年付 49.99 美元的 CN2 GIA-E 限量版，往往补货后十几分钟到半小时就售罄。看到通知第一时间打开下单页，比看十篇攻略都管用。
2. **优惠码提前复制好**：下单页有一个 Promotional Code 输入框，提前把当前有效的优惠码复制到剪贴板，进结算页直接粘贴，省下来的几秒可能就是抢到和抢不到的差别。
3. **支付方式提前准备**：搬瓦工支持支付宝、PayPal、信用卡等，国内用户走支付宝最快。账号里提前绑好支付方式，下单页一路点下去就行。

如果你是第一次接触搬瓦工、又不想等补货，那 CN2 GIA-E **常规版**季付 49.99 美元那款是最稳妥的入门选择，常年有货、12 个机房随意切换、CN2 GIA 三网优化，比抢限量版省心得多。

实操建议是 **Telegram 频道 + 一个第三方监控页**双保险。TG 负责被动接收推送，监控页负责主动确认状态，组合下来基本不会错过补货。

## 十一、买完之后：KiwiVM 后台与机房迁移

下单后账号里会自动出现 KiwiVM 控制面板入口，搬瓦工自研的这套面板比一般 WHMCS 自带的要好用很多。常用的几个功能：

- **Start/Stop/Restart**：开关机、重启。
- **OS Reload**：一键重装系统，支持 AlmaLinux、RockyLinux、CentOS、Debian、Ubuntu、CentOS Stream、Fedora 等主流 Linux 发行版，2026 年新加入了 Ubuntu 26.04 与 Debian 13。
- **Emergency Console**：紧急救援控制台，系统进不去时可以从这里登录排错。
- **rDNS（PTR）**：反向 DNS 设置，做邮件服务器必备。
- **Migrate to another DC**：机房迁移，本文讨论库存时反复提到的就是这个功能。在 CN2 GIA-E 套餐里，你可以一键在 DC6 CN2 GIA-E、DC9 CN2 GIA、JPOS\_1、EUNL\_9 等机房之间切换，迁移过程中数据不丢，几分钟搞定。
- **Snapshots**：快照备份。
- **Audit Log**：操作审计日志，可以看到 VPS 上最近的所有操作记录，方便排查"是谁动了服务器"。
- **API**：完整的 API 接口，进阶用户可以脚本化管理。

机房迁移操作很简单，进 KiwiVM → 找到 Migrate to another DC → 选目标机房 → Confirm → 等几分钟。有两点需要注意：一是迁移会消耗流量（计入当月总流量），二是某些热门机房可能因为满了而无法迁入，这种情况只能等位或换其他机房。
